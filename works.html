Gameboxd API

https://github.com/mpichlmann/Gameboxd-API

  • This is an API webserver built with Python, Flask, SQL Alchemy and Marshmallow. The API connects to a relational database managed with PostgreSQL.
  • The API allows users to register an account, leave reviews for games, and comment on other users reviews. As the API is a REST API, full CRUD functionality is available. The API uses JSON web tokens in order to authenticate users and ensure the correct authorisation before allowing certain functions to be utilised.
  • I thoroughly enjoyed the development of this project as I was able to fully explore and improve upon my knowledge of relational databases, object oriented programming, authentication/authorisation as well as data sanitisation and validation.
  • My understanding of object oriented programming was improved through the repeated use of python objects that interacted with the object relational mapper, SQL Alchemy.
  • The API also helped enhance my understanding of database integrity. By using marshmallow to serialise and deserialise JSON into python objects, I was given a first hand experience in the necessity of data sanitisation, and the importance thereof.
  • Built using Python, VS Code, and Postman along with Flask, SQL Alchemy, Marshmallow, JWT Manager and Bcrypt.
